# 投资者外联（Investor Outreach）

编写简洁、个性化且易于跟进的投资者沟通内容。

## 何时激活

- 向投资者撰写冷启动邮件（Cold Email）时
- 起草请求引荐（Warm Intro）的文案时
- 在会议结束后或对方未回复时发送后续跟进（Follow-ups）
- 在融资过程中编写投资者进展更新（Investor Updates）
- 根据基金投资理念（Thesis）或合伙人的契合点定制外联内容

## 核心原则

1. 个性化每一条外发消息。
2. 保持请求（Ask）的低阻力（易于回复或操作）。
3. 使用事实证据（Proof），而非形容词。
4. 保持简洁。
5. 绝不发送适用于任何投资者的通用模板。

## 冷启动邮件结构（Cold Email Structure）

1. 主题行（Subject Line）：简短且具体。
2. 开篇（Opener）：为什么专门找这位投资者。
3. 核心推介（Pitch）：公司做什么、为什么是现在、有哪些关键证据（Proof）。
4. 请求（Ask）：一个具体的下一步动作。
5. 落款（Sign-off）：姓名、职位，必要时加上一个公信力背书（Credibility Anchor）。

## 个性化来源（Personalization Sources）

参考以下一项或多项：
- 相关的被投组合公司（Portfolio Companies）
- 公开的投资理念、演讲、帖子或文章
- 共同的联系人
- 市场或产品与投资者专注领域的明确契合点

如果缺少这些上下文，请向用户索取，或声明该草稿是等待个性化填充的模板。

## 跟进节奏（Follow-Up Cadence）

默认建议：
- 第 0 天：初始外发
- 第 4-5 天：简短跟进，提供一个新数据点（Data Point）
- 第 10-12 天：最后一次跟进，明确结束（Clean Close）

除非用户要求更长的序列，否则不要在此之后继续催促。

## 引荐请求（Warm Intro Requests）

让引荐人的工作变得轻松：
- 说明为什么这次引荐是契合的
- 包含一段可直接转发的简介（Forwardable Blurb）
- 保持可转发简介在 100 字以内

## 会后更新（Post-Meeting Updates）

应包含：
- 讨论的具体事项
- 承诺的回答或进展更新
- 如果有，提供一个新的证据点（Proof Point）
- 下一步计划

## 质量把控（Quality Gate）

在交付前确认：
- 消息是个性化的
- 请求（Ask）是明确的
- 没有废话或带有“乞求”色彩的语言
- 证据点（Proof Point）是具体的
- 字数保持精简
